# C++ Modules

## CPP00 - _Namespaces, classes, member functions, stdio streams,initialization lists, static, const, and some other basic stuff_

## CPP03 - _Inheritance_

## CPP04 - _Subtype polymorphism, abstract classes, interfaces_
